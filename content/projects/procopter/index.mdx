### Background

Node can be used for a number of applications, so why not use it to fly a drone.

### Summary

Node app that powers a drone flight controller.

### How it works

Parrot has node SDK that can be used to control it's drone. A simple node app can built built to create autonomous drone flights. 

### Take Aways

* Node and Drones are awesome
* Automated flight is easy
* Reliable automate flight is hard.
